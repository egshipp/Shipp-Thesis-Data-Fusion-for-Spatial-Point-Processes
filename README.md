# Shipp-Thesis-Data-Fusion-for-Spatial-Point-Processes
This repository holds the code for the simulation study and data frames used for the simulation results.


File Descriptions:
Sim_Data folder - Holds all simualted data used in for results and discussion in thesis

Data_Fusion_Simulation_Study.R - R script of MCMC algorithm as well as process for simulating the point process

Simulation_Analysis.R - R script for analysis of simulated data and posterior intensity surface estimates
